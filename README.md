## Quick Start

1. Visit [repota.app](https://repota.app)
2. Configure your school details (name, term, academic year, grading system)
3. Add students (manually or bulk import)
4. Enter class scores and exam scores
5. Generate and print A4 report cards

## Tech Stack

- **React 19** + TypeScript
- **TanStack Router** (v1.146+)
- **Tailwind CSS 4** with custom Ghana-themed design tokens
- **Vite 7** for lightning-fast development
- **PWA** with offline support (vite-plugin-pwa)
- **LocalStorage** for data persistence

## Privacy & Security

Your data **NEVER leaves your device**. Everything is stored locally in your browser using LocalStorage. No servers, no cloud, no tracking.
